Один из вариантов запуска модулей - сервисов

View > Tool Windows > Services

Add service > Run Configuration Type > Spring Boot
Выбираем те модули, которые нужно запускать и запускаем

Например при запуске EurekaServerApplication
после запуска мы можем видеть рядом с названием
порт, на котором висит приложение при нажатии на этот
порт - (если вы все правильно сделали до этого)
должна открыться страница Spring Eureka,
типа страницы статистика
со следующими разделами System Status,
Instances currently registered with Eureka (в котором
будут отображаться зарегистрированные сервисы),
General Info и другие
